#CrossOver Video Portal
This simple app describe a Rest Api that contains a video portal with  provided videos, therefore in this portal you have to login with a name and password in order to watch the videos, otherwise you wont have permissions to watch and rate the videos.

Ones you logged correctly you will a list of videos to watch with a rate start based on a mongodb colletions, however, you can select one in order to expand the video and rate it.

For other hand, it has the functionality for logout and clear all the sessions in order to have secure sessions in the application.

The main client application was developed with Angular JS, Node, Express, HTML5 and CSS3, it has a main controller where describes the whole rest request based on the backend on node.
